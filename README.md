# readme

## vue-to-react

### Getting started

```bash
## usage
$ sudo tnpm install @zus/vue-to-react -g

## create your project
## convert 待转换文件路径 可转换文件夹或者单个文件
## -o components 指定输出路径 默认在程序运行目录 react__from__vue 下
$ convert components/test.vue -o components || convert components -o components

```
